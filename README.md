# 🎵 Web Music Player

A simple **browser-based music player** built using HTML, CSS, and JavaScript.  
This project demonstrates **audio playback functionality and interactive UI behavior using JavaScript**.

---

## 🌐 Live Demo

🔗 [Try the Music Player](https://lol-musicplayer.netlify.app)

---

## 🚀 Features

- Play and pause music tracks
- Interactive music player interface
- Track progress bar
- Responsive layout
- Audio playback control using JavaScript

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript

---

## 🎯 Purpose

This project was created to explore:

- JavaScript DOM manipulation
- Working with HTML audio elements
- Building interactive UI components
- Structuring a small front-end web project
